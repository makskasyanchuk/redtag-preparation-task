Library Management App

## Usage

- Deploy metadata to your Salesforce org using the button below named 'Deploy to Salesforce'.
- Assign 'Book Library Permission Set' permission set to your User.
- Open 'Library' App.
- Open 'Home' Tab.
- Use the app functionality.
